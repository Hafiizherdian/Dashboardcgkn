# Dashboardcgkn

This repository contains a Next.js + TypeScript analytics dashboard example (inside `dashboard-app`) using Tailwind CSS and Chart.js (via `react-chartjs-2`).

Quick start:

```bash
cd dashboard-app
npm install
npm install chart.js react-chartjs-2
npm run dev
```

Open http://localhost:3000/dashboard to view the analytics demo.
